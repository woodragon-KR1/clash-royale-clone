## 🎯 변경 이유

### Archer 2 엘릭서로 하향 조정
- **빠른 사이클 유닛**: 저코스트로 자주 사용 가능
- **유연한 대응**: 방어와 반격에 효과적
- **덱 순환**: 원하는 카드를 빠르게 찾을 수 있음

### Giant HP 2800으로 상향
- **탱커 역할 강화**: 5 엘릭서에 걸맞은 내구성
- **타워 공격력 증가 보정**: 강해진 타워에 대응
- **더 긴 생존 시간**: 뒤에서 지원할 시간 확보

### 타워 공격력 증가
- **Princess Tower: 120 DMG** (+50%)
- **King Tower: 150 DMG** (+50%)
- **방어력 강화**: 유닛들이 쉽게 타워를 부수지 못하도록
- **전략적 깊이**: 타워 보호가 더 중요해짐

## 🎮 전략 가이드

### 추천 조합

**1. 빠른 사이클 공격**
- Archer (2) + Archer (2) + Knight (3) = **7 엘릭서**
- 빠르게 압박하며 카드 순환

**2. 중형 푸시**
- Knight (3) + Archer (2) = **5 엘릭서**
- 균형잡힌 공격

**3. 강력한 푸시**
- Giant (5) + Archer (2) = **7 엘릭서**
- 또는 Giant (5) + Knight (3) = **8 엘릭서**
- Giant가 탱킹하며 뒤에서 지원

### 방어 전략

**저비용 방어**
- Archer (2)로 원거리 견제
- Knight (3)로 근접 막기

**타워 의존 방어**
- 타워 사거리 내로 유도
- 향상된 타워 공격력 (120-150)으로 빠른 제거

## ✅ 밸런스 철학

### 코스트 커브
```
2 엘릭서 (Archer) → 빠른 사이클, 유연성
3 엘릭서 (Knight) → 안정성, 균형
5 엘릭서 (Giant)  → 강력한 힘, 게임 체인저
```

### 카운터 관계
- **Giant > 타워**: 높은 HP로 오랜 시간 공격
- **Tower > Archer**: 빠른 제거 (3초)
- **Knight**: 중간 역할, 다목적

### 게임 페이스
- **초반**: Archer(2) 사이클로 압박
- **중반**: Knight(3) + Archer(2) 조합
- **후반**: Giant(5) 대형 푸시

이제 더 균형잡히고 전략적인 게임플레이가 가능합니다! 🎯
